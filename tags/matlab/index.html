¶(4) RLC电路算例

电阻R，电感L和电容C分别与电流源串联组成电路。

电流源电流$i_s(t) = 0.2 exp(-1000t)$

$t=0$时刻，C，L均不储能。 其中电路参数：$C=1\mu F,R=300\Omega,L=2.5mH$

由KCL得：
$C\frac{du}{dt}+\frac{u}{R}+\frac{1}{L}\int^{t}_0u(t)dt = i_s(t)$

电压的一阶导数初始值：$C\frac{u(0+)}{dt}=0.2$

¶a. matlab符号运算

¶b. 数值计算 （采用ode45）

电路电压的二阶微分方程：$C\frac{d2u}{dt2}+\frac{1}{R}\frac{du}{dt}+\frac{1}{L}u(t) = \frac{d i_s(t)}{dt}$

数值计算需要降阶处理：
$y(1) = u; y(2)=u’$, 则设：

$dy(1)=y(2)$

$dy(2)=u’’=\frac{1}{C}[\frac{d i_s(t)}{dt}-\frac{1}{R}\frac{du}{dt}-\frac{1}{L}u] =\frac{1}{C}[\frac{d i_s(t)}{dt}-\frac{1}{R}y(2) - \frac{1}{L}y(1)]$
